=========
Citations
=========

When using FinalCif or any of the software it integrates with, please cite the appropriate
references listed below.


FinalCif
--------

* D. Kratzert, *FinalCif*, https://dkratzert.de/finalcif.html.


CheckCIF and Validation
-----------------------

**IUCr CheckCIF** (online validation service)

* IUCr CheckCIF/PLATON service: https://checkcif.iucr.org/

**PLATON** (offline structure validation, including SQUEEZE)

* A. L. Spek, *Acta Cryst.* **2009**, *D65*, 148–155, doi:`10.1107/S090744490804362X <https://doi.org/10.1107/S090744490804362X>`_.
* A. L. Spek, *Acta Cryst.* **2015**, *C71*, 9–18, doi:`10.1107/S2053229614024929 <https://doi.org/10.1107/S2053229614024929>`_ (SQUEEZE).
* PLATON homepage: https://www.platonsoft.nl/platon/


Libraries Used by FinalCif
---------------------------

**gemmi** (CIF reading and writing)

* P. Wojdyr, *J. Open Source Softw.* **2022**, *7*, 4200, doi:`10.21105/joss.04200 <https://doi.org/10.21105/joss.04200>`_.
* gemmi documentation: https://gemmi.readthedocs.io/en/latest/
